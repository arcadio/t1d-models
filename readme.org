To access materials, download encrypted tarball from the [[https://github.com/arcadio/t1d-models/releases/latest/download/t1d-models.tar.gz.gpg][latest release]].

Decrypt using GnuPG: ~gpg -d t1d-models.tar.gz.gpg >t1d-models.tar.gz~

When prompted for a passphrase, use the string provided with the submission.

Lastly, unpack tarball: ~tar -xf t1d-models.tar.gz~
